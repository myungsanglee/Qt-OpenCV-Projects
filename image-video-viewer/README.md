## 프로젝트 개요
* 이미지 & 비디오 뷰어

## 빌드 방법
### Qt Creator 사용
* Windows
  1. image-video-viewer.pro 수정
  2. For OpenCV in Windows 주석 해제
  3. OpenCV 경로 설정
  4. Run
* Ubuntu
  1. image-video-viewer.pro 수정
  2. For OpenCV in Ubuntu 주석 해제
  3. OpenCV 경로 설정
  4. Run

## 사용 방법
* 첫 화면에서 이미지 뷰어, 비디어 뷰어 중에 하나 클릭
* 뷰어 화면으로 이동하면 Open 버튼 눌러서 파일 열기

## 단축키
키 | 설명
--------- | --------
Backspace | 첫 화면으로 가기
